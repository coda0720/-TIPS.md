<h1 align="center">DPIとPPIの違い</h1>

**DPI**と**PPI**は似ていますが、厳密には別物です。

| 用語 | 意味 | 使う場面 |
| --- | --- | --- |
| **PPI** | **Pixels Per Inch** / 1インチあたりのピクセル数 | 画像データ・画面・Photoshop・Illustrator上の解像度 |
| **DPI** | **Dots Per Inch** / 1インチあたりのインクの点の数 | プリンター・印刷機の出力性能 |

## 簡単に言うと

**PPI = 画像そのものの細かさ**  
**DPI = プリンターがどれだけ細かくインクを打てるか**

印刷用データでは「300DPI必要」と言われることがありますが、画像確認では実質的に 300PPI相当の画像解像度があるか を見ます。

# イラレでPPIを閲覧する方法
画像を選択 → 上部コントロールパネルの「PPI」を見る
ウィンドウ → コントロールにチェックいれると表示

<h1 align="center">印刷用画像のPX・DPI計算方法</h1>

印刷用画像では、次の3つを計算できれば十分です。

```text
1. 必要なpx数を出す
2. 画像を原寸で使ったときのdpiを出す
3. 画像を何cmまで綺麗に使えるか出す
```

---

## 基本の単位変換

DPIは **inch基準** なので、cmやmmをinchに変換します。

```text
1inch = 2.54cm
1inch = 25.4mm
```

### cmをinchにする

```text
cm ÷ 2.54 = inch
```

### mmをinchにする

```text
mm ÷ 25.4 = inch
```

---

# 1. 必要PXを計算する方法

印刷したいサイズと必要dpiが分かっている場合、必要なpx数を計算できます。

## 計算式

```text
印刷サイズcm ÷ 2.54 × 必要dpi = 必要px
```

または、

```text
印刷サイズmm ÷ 25.4 × 必要dpi = 必要px
```

## 例：幅60cmを150dpiで印刷したい場合

```text
60cm ÷ 2.54 = 約23.62inch
23.62inch × 150dpi = 約3543px
```

つまり、幅60cmを150dpiで印刷するには、

```text
横幅 約3543px
```

が必要です。

---

# 2. 画像の実際のDPIを逆算する方法

手元の画像を指定サイズで印刷したとき、何dpi相当になるかを計算できます。

## 計算式

```text
画像px ÷ 印刷サイズinch = 実際のdpi
```

cmで計算する場合は、

```text
画像px ÷ (印刷サイズcm ÷ 2.54) = 実際のdpi
```

## 例：3000pxの画像を幅60cmで印刷する場合

```text
60cm ÷ 2.54 = 約23.62inch
3000px ÷ 23.62inch = 約127dpi
```

つまり、3000pxの画像を幅60cmで印刷すると、

```text
約127dpi
```

になります。

等身大看板なら使える可能性はありますが、近くで見ると少し粗く見える場合があります。

---

# 3. 印刷可能サイズを逆算する方法

画像のpx数が分かっている場合、その画像を何cmまで綺麗に使えるかを計算できます。

## 計算式

```text
画像px ÷ 必要dpi = 印刷可能サイズinch
```

cmに直す場合は、

```text
画像px ÷ 必要dpi × 2.54 = 印刷可能サイズcm
```

mmに直す場合は、

```text
画像px ÷ 必要dpi × 25.4 = 印刷可能サイズmm
```

## 例：3000pxの画像を150dpiで使う場合

```text
3000px ÷ 150dpi = 20inch
20inch × 2.54 = 50.8cm
```

つまり、3000pxの画像は150dpi基準だと、

```text
約50.8cmまで
```

使いやすいです。

## 例：3000pxの画像を300dpiで使う場合

```text
3000px ÷ 300dpi = 10inch
10inch × 2.54 = 25.4cm
```

つまり、3000pxの画像は300dpi基準だと、

```text
約25.4cmまで
```

が目安です。

---

# よく使う計算式まとめ

| やりたいこと       | 計算式                                |
| ------------ | ---------------------------------- |
| 必要pxを出す      | `印刷サイズcm ÷ 2.54 × 必要dpi = 必要px`    |
| 実際のdpiを出す    | `画像px ÷ (印刷サイズcm ÷ 2.54) = 実際のdpi` |
| 印刷可能サイズcmを出す | `画像px ÷ 必要dpi × 2.54 = 印刷可能サイズcm`  |

---

# DPIの目安

| 用途            | 目安       |
| ------------- | -------- |
| Web・画面表示      | 72dpi程度  |
| 遠目で見る大型看板     | 100dpi前後 |
| 等身大看板・イベントパネル | 150dpi前後 |
| 近くで見る展示パネル    | 200dpi前後 |
| 名刺・チラシ・写真印刷   | 300dpi前後 |

---

# 60cm × 180cm の等身大看板に必要なpx目安

| 原寸DPI  | 必要px目安            |
| ------ | ----------------- |
| 72dpi  | 約1701px × 5102px  |
| 100dpi | 約2362px × 7087px  |
| 150dpi | 約3543px × 10630px |
| 200dpi | 約4724px × 14173px |
| 300dpi | 約7087px × 21260px |

---

# まとめ

```text
必要px = 印刷サイズcm ÷ 2.54 × 必要dpi

実際のdpi = 画像px ÷ (印刷サイズcm ÷ 2.54)

印刷可能サイズcm = 画像px ÷ 必要dpi × 2.54
```

印刷では、画像のpx数だけでなく、**実際に何cmで使うか** が重要です。
同じ画像でも、小さく使えば綺麗に見え、大きく引き伸ばすと粗くなります。
