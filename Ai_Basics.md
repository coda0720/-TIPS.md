# Illustrator 基本操作メモ

Illustratorでよく使う操作を、初心者向けに作業順で整理したメモ。
ショートカットは **Mac / Windows** の両方を併記。

---

## 0. 基本のキー対応

| 意味         | Mac                   | Windows            |
| ---------- | --------------------- | ------------------ |
| コピー        | `Command + C`         | `Ctrl + C`         |
| ペースト       | `Command + V`         | `Ctrl + V`         |
| 前面へペースト    | `Command + F`         | `Ctrl + F`         |
| ひとつ前に戻る    | `Command + Z`         | `Ctrl + Z`         |
| やり直し       | `Shift + Command + Z` | `Shift + Ctrl + Z` |
| 等比・水平垂直を保つ | `Shift`               | `Shift`            |
| 中心基準 / 複製  | `Option`              | `Alt`              |

---

## 1. テキスト操作

### 通常の文字入力

1. **文字ツール** を選択する。
2. アートボード上をクリックする。
3. 文字を入力する。
4. フォント、サイズ、行間、字間などを調整する。

---

## 2. パス上文字ツール

### パスに沿って文字を打つ

**パス上文字ツール** を使うと、線や図形のフチに沿って文字を配置できる。

### 手順

1. ペンツール、楕円形ツール、線ツールなどでパスを作る。
2. **文字ツールを長押し** する。
3. **パス上文字ツール** を選択する。
4. 文字を乗せたいパス上をクリックする。
5. 文字を入力する。

### 円や図形の内側に文字を入れる

円の外側ではなく内側に文字を入れたい場合は、パス上文字の **中央ブラケット** を使う。

1. パス上文字を選択する。
2. 文字の開始位置・終了位置・中央位置に出る細い線を確認する。
3. 中央の線をドラッグして、パスの内側へ移動する。
4. 文字が内側に反転する。

うまくいかない場合は、次のメニューから調整する。

```text
書式 > パス上文字 > パス上文字オプション
```

または

```text
Type > Type on a Path > Type on a Path Options
```

### 注意点

* パス上文字は、パスの形を変えると文字の流れも変わる。
* 円の内側に文字を入れると、文字が逆さまに見えることがある。
* 必要に応じて「反転」や「ベースラインシフト」で調整する。

---

## 3. 文字のフチを作る方法

文字のフチを作る方法は主に2つある。

---

### 方法A：アピアランスでフチを作る

文字を編集可能なまま残したい場合は、**アピアランス** を使うのが便利。

### 手順

1. 文字を選択する。
2. `ウィンドウ > アピアランス` を開く。
3. アピアランスパネルで **新規線を追加** する。
4. 線の色と太さを設定する。
5. 線を **塗りより下** に移動する。

### メリット

* 文字を後から打ち直せる。
* 線を複数重ねられる。
* 白フチ、黒フチ、外側の太いフチなどを管理しやすい。

### 例

```text
上：塗り
下：細い白フチ
さらに下：太い黒フチ
```

---

### 方法B：アウトライン化してフチを作る

文字をパス化して、完全に図形として扱いたい場合は **アウトライン作成** を使う。

### 手順

1. 文字を選択する。
2. 次のメニューを選ぶ。

```text
書式 > アウトラインを作成
```

3. 文字がパスに変換される。
4. 次のメニューを選ぶ。

```text
オブジェクト > パス > パスのオフセット
```

5. オフセット値を入力する。
6. 外側に広げたパスを作り、色を設定する。

### 注意点

* アウトライン化すると、文字として編集できなくなる。
* 修正用に、アウトライン化する前の文字レイヤーを複製して残しておくと安全。

---

## 4. 線とパスの考え方

Illustratorの線は、基本的に **パスの中心を基準** に太さが付く。

例えば、線幅を10pxにした場合、中心線から外側に5px、内側に5pxのように見える。

### 注意点

* オープンパス、つまり開いた線では、線を内側・外側に寄せる設定ができないことが多い。
* 図形のようなクローズパスでは、線パネルから線の位置を調整できる場合がある。
* 外側だけにきれいなフチを作りたい場合は、線幅だけでなく **パスのオフセット** を使うと安定する。

---

## 5. アピアランスで複数の線を作る

アピアランスを使うと、1つの文字や図形に複数の線・塗り・効果を重ねられる。

### 手順

1. オブジェクトを選択する。
2. `ウィンドウ > アピアランス` を開く。
3. **新規線を追加** を押す。
4. 線の色・太さを設定する。
5. 必要なら線を複数追加する。
6. アピアランス内で上下順を入れ替える。

### よく使う構成

```text
塗り：文字色
線1：細い白フチ
線2：太い黒フチ
線3：さらに外側の装飾線
```

### ポイント

* 上にある線ほど前面に表示される。
* フチとして使う線は、塗りより下に置くと見やすい。
* アウトライン化しなくても複数フチを作れる。

---

## 6. 整列・中央揃え

複数のオブジェクトをきれいに並べるには **整列パネル** を使う。

### 整列パネルを開く

```text
ウィンドウ > 整列
```

### 中央揃えの手順

1. 中央に揃えたいオブジェクトをすべて選択する。
2. 整列パネルを開く。
3. 必要に応じて基準を選ぶ。

```text
選択範囲に整列
キーオブジェクトに整列
アートボードに整列
```

4. 水平方向中央揃え、垂直方向中央揃えを押す。

### よく使う整列

| 操作       | 用途        |
| -------- | --------- |
| 水平方向中央揃え | 左右の中心を揃える |
| 垂直方向中央揃え | 上下の中心を揃える |
| 左揃え      | 左端を揃える    |
| 右揃え      | 右端を揃える    |
| 上揃え      | 上端を揃える    |
| 下揃え      | 下端を揃える    |
| 分布       | 等間隔に並べる   |

## 7. コピー・前面ペースト

### 同じ位置にコピーする

オブジェクトをまったく同じ位置に重ねて複製したい場合は、**前面へペースト** を使う。

| 操作      | Mac           | Windows    |
| ------- | ------------- | ---------- |
| コピー     | `Command + C` | `Ctrl + C` |
| 前面へペースト | `Command + F` | `Ctrl + F` |

### 手順

1. オブジェクトを選択する。
2. `Command + C` / `Ctrl + C` でコピーする。
3. `Command + F` / `Ctrl + F` で前面にペーストする。

### 用途

* 同じ位置に複製して、色違いを作る。
* フチ用のベースを作る。
* 加工前の形を残しておく。
* マスクや影用のパーツを作る。

---

## 8. 移動・複製・拡大縮小

### 平行移動

| 操作             | Mac            | Windows        |
| -------------- | -------------- | -------------- |
| 水平・垂直・45度方向に移動 | `Shift + ドラッグ` | `Shift + ドラッグ` |

`Shift` を押しながらドラッグすると、水平・垂直・斜め45度方向に動かしやすい。

---

### ドラッグでコピー

| 操作         | Mac                     | Windows              |
| ---------- | ----------------------- | -------------------- |
| 複製しながら移動   | `Option + ドラッグ`         | `Alt + ドラッグ`         |
| 水平・垂直に複製移動 | `Shift + Option + ドラッグ` | `Shift + Alt + ドラッグ` |

### 手順

1. オブジェクトを選択する。
2. `Option` / `Alt` を押しながらドラッグする。
3. コピーが作られる。
4. `Shift` も一緒に押すと、水平・垂直方向にコピーしやすい。

---

### 中心から拡大縮小

| 操作          | Mac                     | Windows              |
| ----------- | ----------------------- | -------------------- |
| 等比拡大縮小      | `Shift + ドラッグ`          | `Shift + ドラッグ`       |
| 中心基準で拡大縮小   | `Option + ドラッグ`         | `Alt + ドラッグ`         |
| 中心基準で等比拡大縮小 | `Shift + Option + ドラッグ` | `Shift + Alt + ドラッグ` |

### 手順

1. オブジェクトを選択する。
2. バウンディングボックスの角をドラッグする。
3. `Shift` で縦横比を固定する。
4. `Option` / `Alt` で中心基準にする。

## 9. 前面・背面の順序変更

Illustratorでは、重なったオブジェクトの表示順を変更できる。

### ショートカット

| 操作   | Mac                   | Windows            |
| ---- | --------------------- | ------------------ |
| 前面へ  | `Command + ]`         | `Ctrl + ]`         |
| 背面へ  | `Command + [`         | `Ctrl + [`         |
| 最前面へ | `Shift + Command + ]` | `Shift + Ctrl + ]` |
| 最背面へ | `Shift + Command + [` | `Shift + Ctrl + [` |

### 訂正ポイント

`Shift + Command + J` は最前面へ移動ではない。
最前面へ移動は次のショートカット。

```text
Mac：Shift + Command + ]
Windows：Shift + Ctrl + ]
```

## 10. リフレクトツール

リフレクトツールを使うと、左右反転・上下反転したコピーを作れる。

### 手順

1. オブジェクトを選択する。
2. ツールバーの **回転ツール** を長押しする。
3. **リフレクトツール** を選択する。
4. オブジェクトを選択した状態で、リフレクトツールをダブルクリックする。
5. 反転方向を選ぶ。

```text
垂直：左右反転
水平：上下反転
```

6. 元を残したい場合は **コピー** を押す。
7. 元を反転させて置き換えたい場合は **OK** を押す。

### メニューから行う場合

```text
オブジェクト > 変形 > リフレクト
```

## 11. ロック・ロック解除

作業中に動かしたくないオブジェクトはロックできる。

### ショートカット

| 操作           | Mac                    | Windows          |
| ------------ | ---------------------- | ---------------- |
| 選択オブジェクトをロック | `Command + 2`          | `Ctrl + 2`       |
| すべてのロックを解除   | `Option + Command + 2` | `Alt + Ctrl + 2` |

### 用途

* 背景を固定する。
* 下絵を固定する。
* 間違えて動かしたくないパーツを固定する。

## 12. ダイレクト選択ツール

ダイレクト選択ツールは、オブジェクト全体ではなく、パスの一部を編集するためのツール。

### できること

* アンカーポイントを個別に動かす。
* 線の一部だけを選択する。
* ハンドルを調整してカーブを変える。
* 図形の一部だけを変形する。

### ショートカット

| ツール        | Mac | Windows |
| ---------- | --- | ------- |
| ダイレクト選択ツール | `A` | `A`     |
| 選択ツール      | `V` | `V`     |

### 使い分け

| ツール        | 役割                  |
| ---------- | ------------------- |
| 選択ツール      | オブジェクト全体を選択・移動・拡大縮小 |
| ダイレクト選択ツール | アンカーポイントやパスの一部を編集   |

---

## 13. ハサミツールでパスを切る

ハサミツールは、パスを途中で切断するツール。

### 手順

1. ハサミツールを選択する。
2. 切りたいパス上、またはアンカーポイントをクリックする。
3. 必要ならもう1か所クリックする。
4. パスが分割される。
5. ダイレクト選択ツールで不要な線を選択する。
6. `Delete` / `Backspace` で削除する。

### ショートカット

| ツール    | Mac | Windows |
| ------ | --- | ------- |
| ハサミツール | `C` | `C`     |

### 注意点

* ハサミツールは、パスを切るだけ。
* 不要な部分を消すには、切った後に選択して削除する。
* ツールバーの赤い斜線は「塗りなし」または「線なし」を意味する。
* 赤い斜線を押しても、オブジェクト自体が削除されるわけではない。

---

## 14. 消しゴムツールとの違い

### 消しゴムツール

消しゴムツールは、ドラッグした部分を削るように消すツール。

### ハサミツール

ハサミツールは、指定した位置でパスを切るツール。

### 違い

| ツール     | 使い方      | 向いている作業     |
| ------- | -------- | ----------- |
| 消しゴムツール | ドラッグして削る | ラフに削りたいとき   |
| ハサミツール  | 点を指定して切る | 正確に切り分けたいとき |

---

## 15. アンカーポイントの平均化

2つ以上のアンカーポイントを、水平・垂直・両方の位置で揃えることができる。

### 手順

1. ダイレクト選択ツールでアンカーポイントを選択する。
2. 次のショートカットを押す。

| 操作  | Mac                    | Windows          |
| --- | ---------------------- | ---------------- |
| 平均化 | `Option + Command + J` | `Alt + Ctrl + J` |

3. 平均化の方向を選ぶ。

```text
水平：Y座標を揃える
垂直：X座標を揃える
両方：同じ位置に重ねる
```

### 用途

* 2点を水平に揃える。
* 2点を垂直に揃える。
* ズレたアンカーポイントを整える。

### 注意点

* 平均化は、選択したポイント自体を動かす。
* 図形の形が変わることがある。
* オブジェクト全体を揃えたい場合は、平均化ではなく **整列パネル** を使う。

---

## 16. パスの連結

切れているパスの端点同士をつなげるには、**連結** を使う。

### ショートカット

| 操作    | Mac           | Windows    |
| ----- | ------------- | ---------- |
| パスの連結 | `Command + J` | `Ctrl + J` |

### 手順

1. ダイレクト選択ツールで、つなげたい2つの端点を選択する。
2. `Command + J` / `Ctrl + J` を押す。
3. パスがつながる。

### 平均化との違い

| 操作  | 役割       |
| --- | -------- |
| 平均化 | 点の位置を揃える |
| 連結  | 端点同士をつなぐ |

## 17. パスファインダーとグループ化の違い

### パスファインダー

複数の図形を合体・型抜き・交差などで、1つの形に変換する機能。

### グループ化

複数のオブジェクトをまとめて扱う機能。
ただし、個別のパスはそのまま残る。

### 違い

| 機能         | 内容            | 個別編集       |
| ---------- | ------------- | ---------- |
| パスファインダー合体 | 複数のパスを1つの形にする | 基本的にしにくくなる |
| グループ化      | 複数のパスをまとめる    | 個別パスを維持できる |

---

### パスファインダーを使う場面

* 重なった図形を1つにしたい。
* 図形を型抜きしたい。
* 複雑な形を作りたい。
* 印刷用・入稿用に形を整理したい。

### グループ化を使う場面

* 複数のパーツをまとめて移動したい。
* 後から個別に修正したい。
* レイヤー内のパーツを整理したい。

### グループ化ショートカット

| 操作     | Mac                   | Windows            |
| ------ | --------------------- | ------------------ |
| グループ化  | `Command + G`         | `Ctrl + G`         |
| グループ解除 | `Shift + Command + G` | `Shift + Ctrl + G` |

---

## 18. ライブペイント

ライブペイントは、線で囲まれたエリアを塗り絵のように塗れる機能。

### 基本手順

1. 線や図形を作る。
2. 塗りたい範囲を作る。
3. 対象を選択する。
4. 次のメニューを選ぶ。

```text
オブジェクト > ライブペイント > 作成
```

5. ライブペイントツールで塗りたい部分をクリックする。

### ショートカット

| ツール        | Mac | Windows |
| ---------- | --- | ------- |
| ライブペイントツール | `K` | `K`     |

### 注意点

* ライブペイントは、必ずしもパスファインダーで合体してから使う必要はない。
* 線で囲まれていれば、ライブペイント化して塗れる場合がある。
* 隙間があると塗れない場合がある。
* 必要に応じて、ギャップオプションで隙間判定を調整する。

### 通常のオブジェクトに戻す

ライブペイントを通常のパスに変換したい場合は、次を使う。

```text
オブジェクト > ライブペイント > 拡張
```

---

## 19. レイヤーとアートボードの違い

### レイヤー

レイヤーは、同じアートボード上で要素を分けて管理するためのもの。

### レイヤーでできること

* 表示 / 非表示を切り替える。
* ロックする。
* 下絵、文字、背景、装飾を分ける。
* バージョン違いを比較する。
* 不要な案を一時的に非表示にする。

### 例

```text
レイヤー1：下絵
レイヤー2：文字
レイヤー3：装飾
レイヤー4：背景
レイヤー5：別案
```

---

### アートボード

アートボードは、同じファイル内にある別ページのようなもの。

### アートボードでできること

* 複数ページのデザインを作る。
* 表面・裏面を分ける。
* サイズ違いのデザインを並べる。
* SNS用、印刷用、バナー用などを同じファイルで管理する。

### 例

```text
アートボード1：名刺 表面
アートボード2：名刺 裏面
アートボード3：SNS用画像
アートボード4：印刷用別案
```

---

### レイヤーとアートボードの使い分け

| 項目   | レイヤー         | アートボード         |
| ---- | ------------ | -------------- |
| 役割   | パーツ管理        | ページ管理          |
| 例    | 文字、背景、装飾     | 表面、裏面、別サイズ     |
| 表示切替 | 得意           | ページ単位ではない      |
| 書き出し | 基本は同じアートボード内 | アートボードごとに書き出せる |

---

## 20. ワークスペースの整理

よく使うパネルは、右側や上部に固定しておくと作業しやすい。

### 表示しておくと便利なパネル

```text
ウィンドウ > パスファインダー
ウィンドウ > レイヤー
ウィンドウ > アピアランス
ウィンドウ > グラデーション
ウィンドウ > リンク
ウィンドウ > 画像トレース
ウィンドウ > 整列
```

### おすすめの配置

| パネル      | 用途           |
| -------- | ------------ |
| パスファインダー | 図形の合体・型抜き    |
| レイヤー     | パーツ管理        |
| アピアランス   | 線・塗り・効果の管理   |
| グラデーション  | 色の変化を作る      |
| リンク      | 配置画像の確認      |
| 画像トレース   | ラスター画像のベクター化 |
| 整列       | 中央揃え・等間隔配置   |

### ワークスペースを保存する

よく使うパネル配置ができたら、ワークスペースとして保存する。

```text
ウィンドウ > ワークスペース > 新規ワークスペース
```

崩れた場合は、保存したワークスペースをリセットする。

```text
ウィンドウ > ワークスペース > リセット
```

### 3. アウトライン化すると文字編集できない

アウトライン化した文字はパスになる。
文字を打ち直せなくなるため、編集用の文字を別レイヤーに残しておくと安全。
