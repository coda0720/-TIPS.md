# クリッピングマスクの基本手順  
**背景画像を、別の画像の形で切り抜く方法**

## 重要
普通の画像2枚をそのまま選択しても、クリッピングマスクはできません。  
**切り抜きに使う画像は、先にトレースして図形に変える必要があります。**

---

## 手順

### 1. 画像を2枚配置する
- 背景にしたい画像
- 切り抜きの形にしたい画像

**ファイル → 配置**

---

### 2. 切り抜き用の画像を図形にする
1. 切り抜き用の画像を選択
2. **画像トレース**
3. **拡張**

これで、切り抜き用画像が図形になります。

---

### 3. 重ね順を確認する
次の順番にします。

- **上**：切り抜き用の図形
- **下**：背景画像

---

### 4. 2つを選択する
- 切り抜き用の図形
- 背景画像

この2つを両方選びます。

---

### 5. クリッピングマスクを作る
**オブジェクト → クリッピングマスク → 作成**

**ショートカット**
- Mac：`Command + 7`
- Windows：`Ctrl + 7`

---

## 覚え方
```text
切り抜き用画像をトレース
→ 拡張
→ 背景画像の上に置く
→ 2つを選択
→ クリッピングマスク

# 画像トレースの手順

1. トレースしたい画像を選択する
2. **オブジェクト → 画像トレース → 作成**
3. **オブジェクト → 画像トレース → 拡張**
4. **オブジェクト → グループ解除**
5. 不要な白い背景を選択して、`Delete`で削除
6. 必要な部分をすべて選択する
7. **オブジェクト → グループ**

## ショートカット

* グループ解除：`Shift + Command + G`
* グループ化：`Command + G`

```text
画像を選択
→ 画像トレース
→ 拡張
→ グループ解除
→ 背景を削除
→ グループ化
```

# Illustrator トレース手順まとめ

Illustratorでのトレースは、画像をそのまま使うのではなく、  
**編集できるベクター画像に変換する作業**です。

基本の流れは以下です。

```text
画像を配置
↓
画像を選択
↓
画像トレース
↓
設定を調整
↓
拡張
↓
グループ解除
↓
不要部分を削除・形を調整
↓
完成
```

---

## 1. トレースとは？

| 種類 | 内容 | 向いているもの |
|---|---|---|
| 画像トレース | PNG / JPGなどを自動でベクター化する | ロゴ、線画、簡単なイラスト |
| 手動トレース | ペンツールで自分でなぞる | 正確に作りたいロゴ、複雑な形 |

## 7. しきい値とは？

**しきい値** は、どこまでを黒として認識するかの設定です。

| しきい値 | 結果 |
|---|---|
| 低い | 黒い部分が少なくなる |
| 高い | 黒い部分が多くなる |

### 使い分け

- 薄い線まで拾いたい場合：**しきい値を上げる**
- 影や汚れまで拾ってしまう場合：**しきい値を下げる**

---


## 9. トレース後は「拡張」する

画像トレースをしただけでは、まだ完全には編集できません。

編集できる図形にするには、上部の **拡張** を押します。

### 拡張とは？

| 状態 | 内容 |
|---|---|
| 拡張前 | 画像トレースのプレビュー状態 |
| 拡張後 | パスや図形として編集できる状態 |

つまり、**拡張して初めて点や線を編集できる** ようになります。

---


## 22. パーツを結合する方法

バラバラになった形を一つにしたい場合は、  
**パスファインダー** を使います。

### 手順

1. 結合したいパーツを複数選択する
2. 上メニューから **ウィンドウ → パスファインダー** を開く
3. **合体** をクリックする

---

## 23. パスを単純化する

画像トレース後は、アンカーポイントが多すぎることがあります。

その場合は、**単純化** を使うと編集しやすくなります。

### 手順

1. オブジェクトを選択する
2. 上メニューから **オブジェクト → パス → 単純化** を選ぶ
3. スライダーで調整する

### 効果

| 効果 | 内容 |
|---|---|
| 点を減らす | 編集しやすくなる |
| データが軽くなる | 動作が軽くなる |
| 形がなめらかになる | ガタつきを減らせる |

ただし、単純化しすぎると形が変わるので注意です。

---

## 24. 手動トレースの基本

自動トレースで綺麗にならない場合は、  
**ペンツール** を使って手動でなぞります。

### 手順

1. 元画像を配置する
2. 元画像の透明度を下げる
3. 元画像レイヤーをロックする
4. 新しいレイヤーを作る
5. ペンツールでなぞる
6. 塗りや線を調整する

## 30. よくある失敗と対処法

| 失敗 | 原因 | 対処法 |
|---|---|---|
| 編集できない | 拡張していない | **拡張** を押す |
| 一部だけ選べない | グループ化されている | **グループ解除** する |
| 白背景が残る | ホワイトを無視していない | **ホワイトを無視** をONにする |
| 線がガタガタ | 元画像の解像度が低い | 高画質画像を使う |
| 点が多すぎる | トレースが細かすぎる | **単純化** を使う |
| 色が多すぎる | 色数設定が多い | 色数を減らす |
| ゴミが残る | ノイズを拾っている | ノイズ値を調整 / 手動で削除 |

---

## 31. 覚えておくべき用語

| 用語 | 意味 |
|---|---|
| ベクター | 拡大しても劣化しにくい図形データ |
| ラスター | JPG / PNGなどの画像データ |
| パス | 点と線で作られた形 |
| アンカーポイント | パスを作る点 |
| ハンドル | 曲線を調整する棒 |
| 拡張 | トレース結果を編集可能なパスに変えること |
| グループ解除 | まとまったパーツをバラバラにすること |
| ダイレクト選択 | 点や線の一部を直接編集すること |

---

## 33. 特に重要なポイント

- **画像トレースだけでは細かく編集できない**
- **編集するには「拡張」が必要**
- **一部だけ選びたい場合は「グループ解除」する**
- **全体を動かすなら選択ツール `V`**
- **点や線を直すならダイレクト選択ツール `A`**
- **白背景を消したい場合は「ホワイトを無視」をON**
- **点が多すぎる場合は「単純化」を使う**
