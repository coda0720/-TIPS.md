# 素材整理
- 素材まとめて音同期
  - レイヤー分けて全選択
  - 音声後同期ならタイムコード維持

- カメラ割り
  - V1メイン（引き）
  - レイヤーごとに作業内容統一

- テキスト
  - ダブルクリックで表示
  - テキストクリックでプロパティ
  - 整列と変形で位置揃え

- オブジェクトマスク
  - Shiftで追加
  - Altで削除
  - ◁▷でトラッキング
  - 選択されてないオブジェクトを不透明に入れてくりぬく
