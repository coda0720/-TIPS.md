4. ***強調ナレ/NA作成中***

 - テロップ挿入/調整
6. テロップ挿入完了（タイムコード解放）
 - ジングル挿入（その他別シーケンス作業分のテンプレコピペ）

# 素材整理
- ジングル
  - 文字左寄り
  - 余白なくす

- 素材まとめて音同期
  - レイヤー分けて全選択
  - 音声後同期ならタイムコード維持

- カメラ割り
  - V1メイン（引き）
  - B-roll用にレイヤーを開けておく
  - カメラ種類ごとにレイヤー統一

- テキスト
  - ダブルクリックで表示
  - テキストクリックでプロパティ
  - 整列と変形で位置揃え
      - 文字入力開始位置の指定も必須

- オブジェクトマスク
  - Shiftで追加
  - Altで削除
  - ◁▷でトラッキング
  - 選択されてないオブジェクトを不透明に入れてくりぬく
  
# HOW TO
### 並列素材のクロスディゾルブ
ディゾルブマーカーをドラッグして素材交差　20fps
### 右側全選択して移動
ショートカット　Ａ
### 話し始め間隔
0.5～1秒の余裕残し

# ラット当て
1. ライブラリフテージ上まとめて右クリック
2. 変更…カラー
3. 上書き [S-log3 S-gamut3.cine]
4. SONY系カメラはこれでカラー統一
