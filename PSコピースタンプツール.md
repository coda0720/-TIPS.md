# Photoshop 2026｜コピースタンプツールの手順

## コピースタンプツールとは

**コピースタンプツール**は、画像の別の場所をコピー元にして、別の場所へ塗るツールです。

例えば、

* 壁のきれいな部分をコピーして汚れを隠す
* 背景の色や質感をコピーして不要物を消す
* 布、床、空、肌などの似た質感を上からなじませる

といった作業に使います。

---

## 基本ショートカット

| 操作         |      Windows |             Mac |
| ---------- | -----------: | --------------: |
| コピースタンプツール |          `S` |             `S` |
| コピー元を指定    | `Alt + クリック` | `Option + クリック` |
| ブラシを小さくする  |          `[` |             `[` |
| ブラシを大きくする  |          `]` |             `]` |
| 自由変形       |   `Ctrl + T` |   `Command + T` |
| ひとつ戻る      |   `Ctrl + Z` |   `Command + Z` |

---

# 基本手順

## 1. 元画像を複製する

まず、失敗しても戻せるように画像レイヤーを複製します。

| 操作      |    Windows |           Mac |
| ------- | ---------: | ------------: |
| レイヤーを複製 | `Ctrl + J` | `Command + J` |

レイヤー構成は以下のようにします。

```text
修正用コピー
元画像
```

---

## 2. コピースタンプツールを選ぶ

左ツールバーから **コピースタンプツール** を選びます。

ショートカット：

```text
S
```

---

## 3. コピー元を指定する

きれいな部分にカーソルを置いて、コピー元を指定します。

| 操作      |      Windows |             Mac |
| ------- | -----------: | --------------: |
| コピー元を指定 | `Alt + クリック` | `Option + クリック` |

例：

```text
Alt + クリック
↓
ここをコピー元として登録
```

---

## 4. 隠したい部分に塗る

消したいゴミや不要部分の上をドラッグして塗ります。

```text
コピー元の質感
↓
隠したい場所へ上塗り
```

---

## 5. コピー元を何度も取り直す

同じ場所からコピーし続けると、模様が繰り返されて不自然になります。

こまめに、

```text
Alt + クリック
```

でコピー元を取り直します。

---

# 「調整あり」と「調整なし」の違い

コピースタンプツールには、上部オプションに **調整あり / 調整なし** があります。
英語版では **Aligned** です。

## 調整あり

**調整あり**にすると、コピー元と塗る場所の位置関係が固定されます。

最初にコピー元を指定して塗り始めると、次に塗る時もコピー元が同じ距離感で追従します。

### 向いている場面

| 向いている作業         | 理由              |
| --------------- | --------------- |
| 壁や床の模様を続けたい     | コピー元の位置関係がズレにくい |
| 線やパターンを自然につなげたい | 同じ流れでコピーできる     |
| 広い範囲をなじませたい     | 連続した質感を保ちやすい    |

### イメージ

```text
コピー元も一緒に移動しながら塗る
```

### 注意点

同じ位置関係でコピーされるため、不要な模様まで一緒に拾うことがあります。

---

## 調整なし

**調整なし**にすると、塗るたびに最初に指定したコピー元へ戻ります。

クリックやドラッグをやめて、もう一度塗り始めると、コピー元が最初の位置から再スタートします。

### 向いている場面

| 向いている作業      | 理由             |
| ------------ | -------------- |
| 同じ色を何度も使いたい  | コピー元が毎回同じ場所に戻る |
| 小さいゴミを点で隠したい | 一定の色や質感を使いやすい  |
| ピンポイント修正     | コピー元がズレにくい     |

### イメージ

```text
毎回、最初のコピー元から塗り直す
```

### 注意点

同じ模様を何度も使いやすいため、繰り返し感が出やすいです。

---

# 調整あり・なしの使い分け

| 設定   | 特徴                 | 向いている作業           |
| ---- | ------------------ | ----------------- |
| 調整あり | コピー元が塗る動きに合わせて追従する | 広い背景、壁、床、布、模様     |
| 調整なし | 毎回同じコピー元から始まる      | 小さいゴミ、点汚れ、同じ色での補修 |

初心者は、まずは以下の使い分けで大丈夫です。

```text
広い範囲 → 調整あり
小さいゴミ → 調整なし
```

---

# サンプルの扱い方

上部オプションに **サンプル** があります。
これは、どのレイヤーをコピー元として参照するかを決める設定です。

## サンプルの種類

| サンプル設定    | 内容                      | 向いている場面            |
| --------- | ----------------------- | ------------------ |
| 現在のレイヤー   | 選択中のレイヤーだけを参照する         | 1枚の画像レイヤーだけで作業する時  |
| 現在のレイヤー以下 | 選択中のレイヤーと、その下のレイヤーを参照する | 新規修正レイヤーで作業する時     |
| すべてのレイヤー  | 表示されている全レイヤーを参照する       | 複数レイヤーを見た目通りに修正する時 |

---

## おすすめ設定

元画像を壊さずに作業するなら、以下がおすすめです。

```text
新規レイヤーを作る
↓
サンプル：現在のレイヤー以下
```

レイヤー構成：

```text
修正用レイヤー
元画像
```

この状態で、修正用レイヤーを選んで作業します。

---

## なぜ「現在のレイヤー以下」がおすすめか

修正用レイヤー自体は空ですが、下にある元画像を見ながらコピーできます。

つまり、

```text
元画像を直接壊さずに
修正だけを別レイヤーに乗せられる
```

という状態になります。

失敗した場合は、修正用レイヤーだけ消せば元に戻せます。

---

# ペーストした部分を回転させたい場合

コピースタンプは、基本的にはドラッグでコピーした質感を塗るツールです。
通常の「貼り付けレイヤー」のように、塗った直後のスタンプだけを毎回自由に回転する操作ではありません。

回転させたい場合は、以下の方法が分かりやすいです。

---

## 方法1：新規レイヤーにスタンプしてから回転する

### 手順

1. 新規レイヤーを作る

| 操作     |            Windows |                   Mac |
| ------ | -----------------: | --------------------: |
| 新規レイヤー | `Ctrl + Shift + N` | `Command + Shift + N` |

2. コピースタンプツールを選ぶ

```text
S
```

3. サンプルを **現在のレイヤー以下** にする

4. コピー元を指定する

| 操作     |      Windows |             Mac |
| ------ | -----------: | --------------: |
| コピー元指定 | `Alt + クリック` | `Option + クリック` |

5. 新規レイヤー上にスタンプする

6. そのスタンプしたレイヤーを選択する

7. 自由変形を使う

| 操作   |    Windows |           Mac |
| ---- | ---------: | ------------: |
| 自由変形 | `Ctrl + T` | `Command + T` |

8. 角の外側にカーソルを置いてドラッグし、回転する

9. `Enter` で確定する

---

## 方法2：選択範囲をコピーして回転する

スタンプというより、別の場所をコピーして貼り付けて回転したい場合はこちらです。

### 手順

1. なげなわツールなどでコピーしたい場所を選択する

2. 選択範囲をコピーして新規レイヤー化する

| 操作              |    Windows |           Mac |
| --------------- | ---------: | ------------: |
| 選択範囲を新規レイヤーへコピー | `Ctrl + J` | `Command + J` |

3. 自由変形する

| 操作   |    Windows |           Mac |
| ---- | ---------: | ------------: |
| 自由変形 | `Ctrl + T` | `Command + T` |

4. 回転・拡大縮小・移動して隠したい場所に合わせる

5. 必要ならレイヤーマスクで境目をなじませる

---

# 回転させる時のショートカットまとめ

| 操作              |    Windows |           Mac |
| --------------- | ---------: | ------------: |
| 自由変形            | `Ctrl + T` | `Command + T` |
| 変形確定            |    `Enter` |      `Return` |
| 変形キャンセル         |      `Esc` |         `Esc` |
| 選択範囲を新規レイヤーへコピー | `Ctrl + J` | `Command + J` |

---

# 作業のおすすめ手順

## 小さいゴミを隠す場合

```text
1. Ctrl + J で元画像を複製
2. S でコピースタンプツール
3. 調整なし
4. Alt + クリックで近くのきれいな部分をコピー
5. ゴミの上を小さく塗る
6. 不自然ならコピー元を取り直す
```

---

## 広い背景をなじませる場合

```text
1. 新規レイヤーを作る
2. サンプルを「現在のレイヤー以下」にする
3. S でコピースタンプツール
4. 調整あり
5. Alt + クリックで近くの背景を指定
6. 少しずつ塗る
7. コピー元を何度か取り直す
```

---

## コピーした部分を回転して使いたい場合

```text
1. コピーしたい部分を選択
2. Ctrl + J で新規レイヤー化
3. Ctrl + T で自由変形
4. 回転して位置を合わせる
5. レイヤーマスクで境目をなじませる
```

---

# よくある失敗

| 失敗           | 原因                 | 対処                           |
| ------------ | ------------------ | ---------------------------- |
| 同じ模様が繰り返される  | コピー元を変えていない        | `Alt + クリック` で何度も取り直す        |
| 色が合わない       | 遠い場所からコピーしている      | 近くの似た色からコピーする                |
| 境目が目立つ       | ブラシが硬すぎる           | ブラシの硬さを下げる                   |
| 修正が元に戻せない    | 元画像に直接作業している       | 新規レイヤーで作業する                  |
| 塗っても何も出ない    | サンプル設定が合っていない      | `現在のレイヤー以下` にする              |
| 別レイヤーの色を拾えない | サンプルが現在のレイヤーになっている | `すべてのレイヤー` か `現在のレイヤー以下` にする |

---

# 初心者向けの基本設定

| 項目     | おすすめ                   |
| ------ | ---------------------- |
| ブラシサイズ | 隠したい部分より少し大きめ          |
| 硬さ     | `0〜30%`                |
| 不透明度   | `70〜100%`              |
| 流量     | `50〜100%`              |
| 調整     | 小さいゴミはなし、広い背景はあり       |
| サンプル   | 新規レイヤー作業なら `現在のレイヤー以下` |

---

# まとめ

| やりたいこと     | 設定・操作                          |
| ---------- | ------------------------------ |
| 小さいゴミを隠す   | 調整なし、近くを `Alt + クリック`          |
| 広い背景をなじませる | 調整あり、コピー元を何度も取り直す              |
| 元画像を壊さず作業  | 新規レイヤー＋サンプル `現在のレイヤー以下`        |
| コピー部分を回転   | `Ctrl + T` / `Command + T`     |
| コピー元を指定    | `Alt + クリック` / `Option + クリック` |

基本は、
**コピー元を近くから取る → 小さく塗る → 不自然ならコピー元を取り直す**
の繰り返しです。
