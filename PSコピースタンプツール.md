# Photoshop｜コピースタンプでゴミを最速削除する方法

コピースタンプツールは、**きれいな部分をコピーして、ゴミの上に塗る機能**です。

---

## 最短手順

1. 元画像レイヤーの上に、**新規レイヤー**を作る
2. `S`で**コピースタンプツール**を選択
3. 画面上部の**サンプル**を
   `現在以下` または `すべてのレイヤー` にする
4. ゴミの近くにある、似た色・模様の場所を探す
5. `Alt`／`Option`を押しながらクリックしてコピー元を指定
6. ゴミの上を短くクリック、または少しずつ塗る
7. 模様が繰り返して見えたら、コピー元を取り直す

Adobe公式でも、`Alt`／`Option＋クリック`でコピー元を指定してから塗る手順が案内されています。

---

| 操作         | Windows            | Mac                   |
| ---------- | ------------------ | --------------------- |
| コピースタンプツール | `S`                | `S`                   |
| コピー元を指定    | `Alt + クリック`       | `Option + クリック`       |
| ブラシを小さくする  | `[`                | `[`                   |
| ブラシを大きくする  | `]`                | `]`                   |
| ブラシを柔らかくする | `Shift + [`        | `Shift + [`           |
| ブラシを硬くする   | `Shift + ]`        | `Shift + ]`           |

| 新規レイヤー     | `Ctrl + Shift + N` | `Command + Shift + N` |
| 画面に合わせて表示  | `Ctrl + 0`         | `Command + 0`         |
| 100％表示     | `Ctrl + 1`         | `Command + 1`         |

## 最速できれいに消すコツ

### 小さなゴミ・ホコリ

1. ゴミより少し大きいブラシにする
2. ゴミのすぐ近くからコピーする
3. ドラッグせず、**1回クリック**で消す

### 長い傷・線

1. 傷の進行方向と似た場所をコピー
2. 一気に塗らず、短く分けて塗る
3. 途中で何度もコピー元を変更する

### 肌・壁・空など滑らかな場所

* 硬さ：`0～30％`
* 不透明度：`30～70％`
* 柔らかいブラシで少しずつ重ねる

### 輪郭・建物・直線付近

* 硬さ：`70～100％`
* 小さめのブラシを使う
* 輪郭と同じ方向からコピーする

---

## おすすめ設定

画面上部のオプションバーを次のようにします。

| 設定   | おすすめ     |
| ---- | -------- |
| モード  | 通常       |
| 不透明度 | 100％から開始 |
| 流量   | 100％     |
| サンプル | 現在以下     |
| 調整あり | 基本はオン    |

### 「調整あり」の違い

* **オン**
  コピー元と塗る場所の距離関係を保ったまま移動する
  → 通常はこちら

* **オフ**
  マウスを離すたびに、最初のコピー位置から再開する
  → 同じ模様を何度も置きたい場合に使用

## 元画像を壊さない方法

1. 元画像の上に新規レイヤーを作る
2. 新規レイヤーを選択
3. サンプルを`現在以下`にする
4. 新規レイヤーへ修正を描く

失敗しても、修正レイヤーを消せば元に戻せます。

---

## うまく消えないとき

| 症状              | 対処                    |
| --------------- | --------------------- |
| 何も描けない          | サンプルを`現在以下`にする        |
| 不自然な丸が見える       | ブラシを柔らかくする            |
| 同じ模様が繰り返される     | コピー元を頻繁に変える           |
| 色が合わない          | ゴミのすぐ近くからコピーする        |
| 輪郭が崩れる          | ブラシを小さく・硬くする          |
| 修正が濃すぎる         | 不透明度を下げる              |
| スマートオブジェクトに描けない | 新規レイヤーを作り、`現在以下`で作業する |

---

## コピー元を回転・拡大したい場合

1. `Alt`／`Option + クリック`でコピー元を指定
2. **［ウィンドウ］→［コピーソース］**
3. 回転角度、幅、高さ、反転を調整
4. プレビューを見ながら塗る

コピーソースパネルでは、コピー元の回転・拡大縮小・反転・位置調整ができます。

---

## 最短で覚える操作

```text
S
↓
Alt／Option＋クリック
↓
ゴミをクリック
↓
不自然ならコピー元を取り直す
```

基本は、**近くからコピーして、小さく短く塗る**だけで十分です。
