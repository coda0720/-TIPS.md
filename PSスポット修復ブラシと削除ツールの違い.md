# Photoshop 2026｜スポット修復ブラシと削除ツールの違い・使い分け

## 目的

画像内のゴミ、点汚れ、不要物を消すときに使う代表的なツールは、主にこの2つです。

| ツール          | 目的          | 向いているもの              |
| ------------ | ----------- | -------------------- |
| スポット修復ブラシツール | 小さい汚れを自然に消す | 点ゴミ、ホコリ、肌の小さなシミ、細かい傷 |
| 削除ツール        | 大きめの不要物を消す  | 小物、人物、線、写り込み、背景の邪魔な物 |

---

# 1. スポット修復ブラシツールとは

**スポット修復ブラシツール**は、クリックまたはドラッグした部分を、周囲になじませて消すツールです。

小さいゴミをすばやく消したい時に向いています。

## 向いている作業

| 向いているもの | 例           |
| ------- | ----------- |
| 点のゴミ    | ホコリ、黒点、白点   |
| 小さい汚れ   | 机の小さな汚れ、壁の点 |
| 肌の補正    | シミ、ニキビ、細かい傷 |
| 細い傷     | スキャン画像の小さな線 |

---

## スポット修復ブラシの手順

### 1. 元画像を複製する

元画像を直接編集しないために、まず複製します。

| 操作      |    Windows |           Mac |
| ------- | ---------: | ------------: |
| レイヤーを複製 | `Ctrl + J` | `Command + J` |

レイヤー構成：

```text
修正用コピー
元画像
```

---

### 2. スポット修復ブラシツールを選ぶ

左ツールバーから **スポット修復ブラシツール** を選びます。

ショートカット：

```text
J
```

※ `J` を押して別の修復系ツールが出る場合は、左ツールバーの `J` 系ツールを長押しして、スポット修復ブラシを選びます。

---

### 3. ブラシサイズを調整する

消したいゴミより、少し大きいブラシサイズにします。

| 操作        | Windows / Mac |
| --------- | ------------: |
| ブラシを小さくする |           `[` |
| ブラシを大きくする |           `]` |

おすすめ：

```text
ゴミより少し大きめ
硬さ：0〜50%
```

---

### 4. ゴミをクリック、またはなぞる

小さい点ならクリック。
細長い汚れならドラッグしてなぞります。

```text
点ゴミ → クリック
線や傷 → ドラッグ
```

---

## スポット修復ブラシのコツ

| コツ             | 内容                         |
| -------------- | -------------------------- |
| 小さく直す          | 一気に広範囲を消さない                |
| ブラシは少し大きめ      | ゴミより少し大きくする                |
| 不自然なら戻す        | `Ctrl + Z` / `Command + Z` |
| 同じ場所を何度もこすらない  | ぼやけやすくなる                   |
| 背景が単純なほど成功しやすい | 壁、空、机など                    |

---

# 2. 削除ツールとは

**削除ツール**は、不要な物をブラシでなぞって、Photoshopが周囲を判断して消してくれるツールです。

スポット修復ブラシよりも、やや大きい不要物に向いています。

## 向いている作業

| 向いているもの | 例          |
| ------- | ---------- |
| 大きめの不要物 | 小物、ゴミ、紙くず  |
| 人物の除去   | 背景に写った人    |
| 線状の邪魔物  | 電線、傷、ひも    |
| 背景の整理   | 写り込み、不要な模様 |

---

## 削除ツールの手順

### 1. 元画像を複製する

まず元画像を複製します。

| 操作      |    Windows |           Mac |
| ------- | ---------: | ------------: |
| レイヤーを複製 | `Ctrl + J` | `Command + J` |

レイヤー構成：

```text
削除作業用
元画像
```

---

### 2. 削除ツールを選ぶ

左ツールバーから **削除ツール / Remove Tool** を選びます。

ショートカット：

```text
J
```

※ `J` 系ツールの中にあります。
スポット修復ブラシと同じグループに入っている場合があります。

---

### 3. ブラシサイズを調整する

削除したい物より、少し大きめにします。

| 操作        | Windows / Mac |
| --------- | ------------: |
| ブラシを小さくする |           `[` |
| ブラシを大きくする |           `]` |

---

### 4. 消したい部分をなぞる

不要物をブラシで塗ります。

```text
不要物全体を軽くなぞる
↓
Photoshopが周囲を使って自動補完する
```

削除ツールでは、対象を囲むように塗るより、**消したい物をしっかり覆う**ように塗ると成功しやすいです。

---

### 5. 結果が不自然ならやり直す

一度で綺麗に消えない場合があります。

その場合は、

```text
Ctrl + Z
↓
ブラシサイズを変える
↓
塗る範囲を少し変える
↓
もう一度試す
```

と調整します。

---

# 3. スポット修復ブラシと削除ツールの違い

| 比較       | スポット修復ブラシ   | 削除ツール         |
| -------- | ----------- | ------------- |
| 得意なサイズ   | 小さい         | 小〜中、大きめ       |
| 操作       | クリック・なぞる    | なぞって削除        |
| 向いているもの  | 点ゴミ、シミ、細かい傷 | 不要物、人物、線、写り込み |
| 自然さ      | 小さい補正に強い    | 大きめの削除に強い     |
| 失敗しやすい場面 | 複雑な模様の上     | 背景が複雑すぎる場所    |
| 作業感      | 細かい修正向き     | ざっくり削除向き      |

# 4. 使い分け早見表

| 状況           | おすすめツール                        |
| ------------ | ------------------------------ |
| 小さい黒点を消したい   | スポット修復ブラシ                      |
| 肌の小さなシミを消したい | スポット修復ブラシ                      |
| 壁の小さい汚れを消したい | スポット修復ブラシ                      |
| 写り込んだ小物を消したい | 削除ツール                          |
| 背景の人物を消したい   | 削除ツール                          |
| 電線や細い線を消したい  | 削除ツール                          |
| 消した後に少し不自然   | コピースタンプで仕上げ                    |
| 複雑な背景で崩れる    | 選択範囲＋コンテンツに応じた塗りつぶし、またはコピースタンプ |

# 5. 基本のおすすめ作業順

初心者は、次の順番で試すと分かりやすいです。

```text
1. 元画像を複製
2. 小さいゴミはスポット修復ブラシ
3. 大きい不要物は削除ツール
4. 不自然な部分をコピースタンプで整える
5. PSDで保存
```

---

# 6. スマートオブジェクトに適用できない理由

スマートオブジェクトは、元画像を保護したまま扱うためのレイヤーです。

そのため、スポット修復ブラシや削除ツールのように、画像のピクセルを直接塗り替えるツールは、そのままでは使えない場合があります。

簡単に言うと、

```text
スマートオブジェクト
＝ 元画像を保護している箱

スポット修復ブラシ・削除ツール
＝ ピクセルを直接修正する道具
```

なので、箱の外側から中身のピクセルを直接塗れない、というイメージです。

---

# 7. スマートオブジェクトでツールが使えない時の対処法

## 対処法A：ラスタライズする

スマートオブジェクトを通常の画像レイヤーに変換する方法です。

### 手順

1. レイヤーパネルでスマートオブジェクトを選ぶ
2. 右クリック
3. **レイヤーをラスタライズ**
4. スポット修復ブラシや削除ツールを使う

または、

```text
レイヤー
↓
ラスタライズ
↓
スマートオブジェクト
```

## メリット

| メリット      | 内容              |
| --------- | --------------- |
| すぐ編集できる   | 修復ブラシや削除ツールが使える |
| 操作が分かりやすい | 普通の画像レイヤーとして扱える |
| 初心者向き     | エラーが少ない         |

## デメリット

| デメリット              | 内容             |
| ------------------ | -------------- |
| 元のスマートオブジェクト性がなくなる | 後から元データを編集しにくい |
| 拡大縮小に弱くなる          | 何度も変形すると劣化しやすい |
| 非破壊編集ではなくなる        | 直接編集になる        |

---

## 対処法B：同じ画像を複製してからラスタライズする

おすすめはこちらです。

### 手順

1. スマートオブジェクトレイヤーを複製する

| 操作      |    Windows |           Mac |
| ------- | ---------: | ------------: |
| レイヤーを複製 | `Ctrl + J` | `Command + J` |

2. 複製したレイヤーを選ぶ
3. 右クリック
4. **レイヤーをラスタライズ**
5. ラスタライズした複製レイヤーで修正する

レイヤー構成：

```text
修正用レイヤー（ラスタライズ済み）
元のスマートオブジェクト
```

## この方法が安全な理由

元のスマートオブジェクトを残したまま、上のコピーだけを修正できます。

つまり、

```text
元データは保護
修正作業はコピー側
```

になります。

失敗したら、修正用レイヤーだけ削除すればやり直せます。

---

## 対処法C：新規空レイヤーを作って作業する

ツールの設定でサンプルを使える場合は、新規レイヤー上で修正できます。

### 手順

1. スマートオブジェクトの上に新規レイヤーを作る

| 操作     |            Windows |                   Mac |
| ------ | -----------------: | --------------------: |
| 新規レイヤー | `Ctrl + Shift + N` | `Command + Shift + N` |

2. レイヤー名を「修正用」にする
3. スポット修復ブラシ、または修復系ツールを選ぶ
4. 上部オプションで **全レイヤーを対象**、または **現在のレイヤー以下** を選ぶ
5. 新規レイヤーを選んだ状態で修正する

レイヤー構成：

```text
修正用レイヤー
スマートオブジェクト
```

## メリット

| メリット           | 内容             |
| -------------- | -------------- |
| 元画像を壊さない       | 修正だけ別レイヤーにできる  |
| やり直しやすい        | 修正用レイヤーを消せば戻せる |
| スマートオブジェクトを残せる | 元データを保護できる     |

## 注意点

ツールや設定によっては、新規空レイヤー上ではうまく適用されない場合があります。

その場合は、

```text
サンプル設定を確認
↓
それでも無理なら複製してラスタライズ
```

の順で対処します。

---

# 8. サンプル設定の確認

修復系ツールには、上部オプションに **サンプル** や **全レイヤーを対象** の設定がある場合があります。

これは、どのレイヤーの情報を使って修正するかを決める設定です。

| サンプル設定    | 内容                 | 向いている場面            |
| --------- | ------------------ | ------------------ |
| 現在のレイヤー   | 選択中のレイヤーだけを見る      | 通常レイヤーを直接修正する時     |
| 現在のレイヤー以下 | 選択中のレイヤーと下のレイヤーを見る | 新規修正レイヤーで作業する時     |
| すべてのレイヤー  | 表示中の全レイヤーを見る       | 複数レイヤーを見た目通りに修正する時 |
| 全レイヤーを対象  | 表示されているレイヤーを参照する   | 空レイヤーに修正を乗せたい時     |

---

# 9. 「適用できない」と出た場合の対処法

## 状況1：スマートオブジェクトなので編集できない

### 原因

スマートオブジェクトは、直接ピクセル編集できない場合があります。

### 対処

おすすめ順：

```text
1. 新規レイヤーを作って「全レイヤーを対象」で試す
2. スマートオブジェクトを複製する
3. 複製した方だけラスタライズする
4. ラスタライズ済みレイヤーで修正する
```

---

## 状況2：ブラシで塗っても何も起きない

### 原因候補

| 原因               | 対処                               |
| ---------------- | -------------------------------- |
| 選択しているレイヤーが違う    | 修正したいレイヤーを選ぶ                     |
| スマートオブジェクトを選んでいる | 複製してラスタライズ、または新規レイヤー作業           |
| サンプル設定が違う        | `全レイヤーを対象` や `現在のレイヤー以下` にする     |
| レイヤーがロックされている    | ロックを解除する                         |
| 選択範囲が残っている       | `Ctrl + D` / `Command + D` で選択解除 |
| ブラシサイズが小さすぎる     | `]` で大きくする                       |
| 不透明度や流量が低い       | 上部オプションで上げる                      |

---

## 状況3：修正結果が不自然

### 原因候補

| 原因          | 対処                 |
| ----------- | ------------------ |
| 一度に広く消しすぎ   | 小分けに作業する           |
| 背景が複雑       | 削除ツール後にコピースタンプで整える |
| ブラシが硬すぎる    | 硬さを下げる             |
| 参照範囲が合っていない | サンプル設定を変える         |
| 似た背景が少ない    | 別の方法を使う            |

---

# 10. ラスタライズするべきか、複製するべきか

## そのままラスタライズしてよい場合

| 状況          | 理由                  |
| ----------- | ------------------- |
| もう拡大縮小しない   | スマートオブジェクトを残す必要が少ない |
| その画像だけで完結する | 元データの再編集が不要         |
| 作業が簡単       | すぐ修正したい時向き          |
