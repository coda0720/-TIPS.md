# Photoshop 2026：切り抜きの基本手順

## 1. 切り抜きの考え方

Photoshopでの切り抜きは、基本的に以下の流れで行います。

```text
画像を開く
↓
切り抜きたい対象を選択する
↓
選択範囲を作る
↓
レイヤーマスクを追加する
↓
不要な部分を見えなくする
↓
ブラシで細かく調整する
↓
PNGなどで書き出す
```

---

## 2. 切り抜きで使う主な機能

| 機能 | 用途 |
|---|---|
| 選択範囲 | 切り抜きたい場所を指定する |
| 自動選択ツール | 似た色の範囲を自動で選択する |
| クイック選択ツール | なぞった範囲を自動で広げて選択する |
| オブジェクト選択ツール | 人物や物体を自動認識して選択する |
| レイヤーマスク | 画像を消さずに一部だけ隠す |
| ブラシツール | マスクを手動で調整する |
| 画面サイズに合わせる | 作業画面全体を見やすくする |

# 方法A：選択範囲で囲んで切り抜く方法

## 4. 選択範囲で切り抜く基本手順

人物、商品、ロゴなどを大まかに囲んで切り抜く方法です。

### 手順

1. Photoshopで画像を開く
2. レイヤーパネルで画像レイヤーを選択する
3. 必要なら `Ctrl + J` / `Command + J` でレイヤーを複製する
4. `投げなわツール`、`長方形選択ツール`、`オブジェクト選択ツール` などで対象を囲む
5. 切り抜きたい範囲が選択されていることを確認する
6. レイヤーパネル下部の `レイヤーマスクを追加` をクリックする
7. 選択範囲の内側だけが表示される
8. 選択範囲の外側は見えなくなる

---

## 5. レイヤーマスクで切り抜く理由

レイヤーマスクは、画像を実際に削除せずに、見える部分と見えない部分を分ける機能です。

つまり、失敗してもあとから修正できます。

| 操作 | 結果 |
|---|---|
| 消しゴムで消す | 画像そのものが削られる |
| レイヤーマスクで隠す | 画像は残ったまま見えなくなる |
| マスクを修正する | あとから表示・非表示を調整できる |

---

# 方法B：自動選択ツールで背景側を選んで切り抜く方法

## 6. 自動選択ツールとは

自動選択ツールは、クリックした場所と近い色の範囲を自動で選択するツールです。

白背景、単色背景、色がはっきり分かれている画像に向いています。

---

## 7. 自動選択ツールで範囲外を選択する手順

背景を選んでから、背景を見えなくする方法です。

### 手順

1. Photoshopで画像を開く
2. レイヤーパネルで画像レイヤーを選択する
3. `Ctrl + J` / `Command + J` でレイヤーを複製する
4. ツールバーから `自動選択ツール` を選ぶ  
   ※ ショートカットは `W`
5. 背景部分をクリックする
6. 背景が選択される
7. 背景が一度で選びきれない場合は、`Shift` を押しながら追加でクリックする
8. 余計な部分まで選ばれた場合は、`Alt` / `Option` を押しながらクリックして選択範囲から外す
9. 背景だけが選択されている状態にする
10. `Ctrl + Shift + I` / `Command + Shift + I` で選択範囲を反転する
11. 対象物側が選択される
12. レイヤーパネル下部の `レイヤーマスクを追加` をクリックする
13. 対象物だけが表示され、背景が見えなくなる

---

## 8. 自動選択ツールの設定

| 設定 | 内容 |
|---|---|
| 許容値 | どれくらい近い色まで選ぶか |
| 隣接 | チェックあり：つながった範囲だけ選ぶ |
| 隣接 | チェックなし：画像全体から近い色を選ぶ |
| アンチエイリアス | 境界を少しなめらかにする |
| 全レイヤーを対象 | 表示中の複数レイヤーを参考に選択する |

### 許容値の目安

| 許容値 | 向いている場面 |
|---|---|
| 低い | 色の差がはっきりしている |
| 高い | 背景に少しムラがある |
| 高すぎる | 対象物まで選ばれやすいので注意 |

# 方法C：レイヤーマスクを使って選択範囲外を見えなくする方法

## 9. レイヤーマスクの基本

レイヤーマスクでは、白・黒・グレーで表示範囲を調整します。

| マスクの色 | 結果 |
|---|---|
| 白 | 見える |
| 黒 | 見えない |
| グレー | 半透明になる |

---

## 10. 選択範囲からレイヤーマスクを作る手順

1. 切り抜きたい対象を選択する
2. 対象の周りに点線が出ていることを確認する
3. レイヤーパネルで画像レイヤーを選択する
4. レイヤーパネル下部の `レイヤーマスクを追加` をクリックする
5. 選択範囲の内側が白、外側が黒のマスクになる
6. 白い部分だけが表示される
7. 黒い部分は見えなくなる

---

## 11. 選択範囲外を見えなくする考え方

選択範囲を作った状態でレイヤーマスクを追加すると、基本的に選択範囲の中だけが表示されます。

```text
選択範囲の内側 = 表示
選択範囲の外側 = 非表示
```

もし逆になった場合は、マスクを反転します。

| 操作 | Windows | Mac |
|---|---:|---:|
| マスクを反転 | `Ctrl + I` | `Command + I` |

注意：`Ctrl + I` / `Command + I` は、マスクのサムネイルを選択している状態で使います。

---

# 方法D：ブラシでマスクを調整する方法

## 12. ブラシで見えない部分を増やす

レイヤーマスクを使うと、ブラシで塗った場所をあとから見えなくできます。

### 手順

1. レイヤーパネルで `レイヤーマスクのサムネイル` をクリックする
2. 白黒の枠がマスク側についていることを確認する
3. `B` キーでブラシツールを選ぶ
4. 描画色を黒にする
5. 画像上を塗る
6. 黒で塗った部分が見えなくなる

### 結果

```text
黒で塗る = 隠す
```

---

## 13. ブラシで見える部分を戻す

消しすぎた場合は、白で塗ると戻せます。

### 手順

1. レイヤーマスクのサムネイルを選択する
2. `B` キーでブラシツールを選ぶ
3. 描画色を白にする
4. 戻したい場所を塗る
5. 白で塗った部分が再び見える

### 結果

```text
白で塗る = 戻す / 見せる
```

---

## 14. ブラシ調整で使うショートカット

| 操作 | Windows | Mac |
|---|---:|---:|
| ブラシツール | `B` | `B` |
| 描画色と背景色を入れ替え | `X` | `X` |
| 描画色を白黒に戻す | `D` | `D` |
| ブラシサイズを小さくする | `[` | `[` |
| ブラシサイズを大きくする | `]` | `]` |
| ブラシを柔らかくする | `Shift + [` | `Shift + [` |
| ブラシを硬くする | `Shift + ]` | `Shift + ]` |

### 覚え方

```text
D = 白黒に戻す
X = 白と黒を入れ替える
黒 = 隠す
白 = 見せる
```

# 18. 切り抜き後に透明PNGで書き出す方法

## 手順

1. 背景が市松模様になっていることを確認する
2. 不要な背景レイヤーを非表示にする
3. `ファイル` → `書き出し` → `書き出し形式...` をクリック
4. 形式を `PNG` にする
5. `透明部分` をオンにする
6. 必要に応じてサイズを変更する
7. `書き出し` をクリックする

---

# 19. 切り抜き作業のおすすめ手順

```text
画像を開く
↓
元レイヤーを複製
↓
画面サイズに合わせる
↓
自動選択ツールや投げなわツールで選択範囲を作る
↓
必要なら選択範囲を反転
↓
レイヤーマスクを追加
↓
ブラシでマスクを微調整
↓
透明PNGで書き出す
```

# 20. よくある失敗と対処法

| 失敗 | 原因 | 対処 |
|---|---|---|
| 背景ではなく対象物が消えた | 選択範囲が逆 | マスクを選択して `Ctrl + I` / `Command + I` |
| ブラシで塗っても反応しない | 画像本体を選択している | レイヤーマスクのサムネイルを選択する |
| 黒で塗っているのに消えない | 描画色が黒ではない | `D` → `X` で白黒を切り替える |
| 白背景が残る | 背景レイヤーが表示されている | 背景レイヤーを非表示にする |
| PNGなのに透明にならない | 透明部分がオフ | 書き出し時に `透明部分` をオン |
| 境界がガタガタ | 選択範囲が粗い | マスクをブラシで調整する |
| 消しすぎた | 黒で塗りすぎた | 白で塗って戻す |

---

# 21. まとめ

| やりたいこと | 操作 |
|---|---|
| 背景を自動で選びたい | 自動選択ツール |
| 対象物を選びたい | オブジェクト選択ツール / クイック選択ツール |
| 背景を選んでしまった | 選択範囲を反転 |
| 選択範囲外を見えなくしたい | レイヤーマスクを追加 |
| 一部を見えなくしたい | マスクを黒で塗る |
| 消しすぎた部分を戻したい | マスクを白で塗る |
| 全体を画面に収めたい | `Ctrl + 0` / `Command + 0` |
| 透明PNGで出したい | PNG + 透明部分オン |

---

# 22. 覚え方

```text
選択範囲 = 残す場所を決める
レイヤーマスク = 消さずに隠す
黒で塗る = 隠す
白で塗る = 見せる
背景を選んだら = 反転してからマスク
全体を見る = Ctrl + 0 / Command + 0
透明PNG = PNG + 透明部分オン
```
