## 6. 一文字だけ色やサイズを変える手順

一文字だけ色・サイズ・フォントを変えたい場合は、文字ツールでその文字だけを選択します。

### 手順

1. レイヤーパネルでテキストレイヤーを選ぶ
2. `T` キーで文字ツールを選ぶ
3. 文字の上をクリックする
4. 変更したい一文字だけをドラッグして選択する
5. 上部オプションバー、または `文字パネル` で設定を変える
6. `Ctrl + Enter` / `Command + Return` で確定する

### 変更できるもの

| 項目 | 内容 |
|---|---|
| 文字色 | 一文字だけ赤くするなど |
| 文字サイズ | 一文字だけ大きくする |
| フォント | 一文字だけ別フォントにする |
| 太さ | 一文字だけ太字にする |
| ベースライン | 一文字だけ上下にずらす |
| 字間 | 一文字の前後の間隔を変える |

# 文字間隔の調整

## 8. カーニングとトラッキングの違い

| 項目 | 内容 | 使う場面 |
|---|---|---|
| カーニング | 特定の2文字間の間隔 | 「A」と「V」の間だけ詰めたい |
| トラッキング | 選択した文字全体の間隔 | タイトル全体を広げたい |
| 行送り | 行と行の間隔 | 複数行の読みやすさ調整 |

---

## 9. 文字間隔を調整する手順

### 全体の文字間隔を変える場合

## Alt (Command) + →/←
ｰ選択間隔の調整ショートカット

1. レイヤーパネルでテキストレイヤーを選ぶ
2. `T` キーで文字ツールを選ぶ
3. 調整したい文字全体を選択する
4. `文字パネル` を開く
5. `トラッキング` の数値を変更する
6. `Ctrl + Enter` / `Command + Return` で確定する

### 一部だけ文字間隔を変える場合

1. `T` キーで文字ツールを選ぶ
2. 間隔を変えたい文字だけを選択する
3. `文字パネル` の `トラッキング` を変更する
4. 必要ならカーニングも調整する
5. 確定する

# 文字の変形

## 11. 自由変形で文字を拡大・縮小・回転する

テキストレイヤーは、文字の内容を保ったまま自由変形できます。

### 手順

1. レイヤーパネルでテキストレイヤーを選ぶ
2. `Ctrl + T` / `Command + T` を押す
3. 四隅のハンドルをドラッグして拡大・縮小する
4. 外側にカーソルを置いてドラッグすると回転できる
5. `Enter` / `Return` で確定する

---

## 13. ワープテキストを使う方法

### 手順

1. レイヤーパネルでテキストレイヤーを選ぶ
2. `T` キーで文字ツールを選ぶ
3. 上部オプションバーの `ワープテキストを作成` をクリックする
4. スタイルを選ぶ  
   例：円弧、下弦、上弦、旗、波形
5. カーブ、水平ゆがみ、垂直ゆがみを調整する
6. `OK` をクリックする

# レイヤースタイル

## 15. レイヤースタイルでよく使う効果

| 効果 | 内容 | 用途 |
|---|---|---|
| ドロップシャドウ | 影をつける | 文字を背景から浮かせる |
| 境界線 | 文字に縁取りをつける | 読みやすくする |
| 光彩 外側 | 外側に光を出す | ネオン風、発光文字 |
| 光彩 内側 | 内側に光を入れる | 立体感 |
| グラデーションオーバーレイ | 文字にグラデーションをかける | ロゴ風 |
| カラーオーバーレイ | 文字全体に色を重ねる | 色変更 |
| ベベルとエンボス | 立体感をつける | 金属文字、タイトル |

---

## 16. レイヤースタイルを使う手順

1. レイヤーパネルでテキストレイヤーを選ぶ
2. レイヤーパネル下部の `fx` をクリックする
3. 使いたい効果を選ぶ  
   例：`境界線`、`ドロップシャドウ`
4. 設定画面で数値を調整する
5. `OK` をクリックする

# 文字切り抜き

|---|---|
| クリッピングマスク | 文字の形の中に画像を表示する |
| レイヤーマスク | 文字の形で一部を隠す |
| 選択範囲化 | 文字の形を選択範囲にする |

## 20. 文字の中に画像を入れる方法

文字の形の中だけに写真や模様を表示する方法です。

### レイヤー構造

```text
写真レイヤー
テキストレイヤー
背景レイヤー
```

### 手順

1. 背景レイヤーを用意する
2. テキストレイヤーを作る
3. テキストレイヤーの上に写真レイヤーを置く
4. 写真レイヤーを右クリックする
5. `クリッピングマスクを作成` をクリックする
6. 文字の形の中だけに写真が表示される

### ショートカット

| 操作 | Windows | Mac |
|---|---:|---:|
| クリッピングマスク作成 | `Alt + Ctrl + G` | `Option + Command + G` |

---

## 21. 文字の形で画像を切り抜く方法

文字の形を選択範囲にして、画像を切り抜く方法です。

### 手順

1. テキストレイヤーを作る
2. 画像レイヤーを用意する
3. `Ctrl` / `Command` を押しながら、テキストレイヤーのサムネイルをクリックする
4. 文字の形が選択範囲になる
5. 画像レイヤーを選択する
6. レイヤーパネル下部の `レイヤーマスクを追加` をクリックする
7. 文字の形だけ画像が表示される

### ポイント

この方法なら画像自体を消さずに、文字の形で見せることができます。

---

## 22. 背景から文字を抜いたように見せる方法

背景画像から文字部分をくり抜いたように見せたい場合は、文字の選択範囲を使ってマスクします。

### 手順

1. 背景画像レイヤーを用意する
2. テキストレイヤーを作る
3. `Ctrl` / `Command` を押しながらテキストレイヤーのサムネイルをクリックする
4. 文字の形の選択範囲ができる
5. 背景画像レイヤーを選択する
6. レイヤーマスクを追加する
7. 必要なら `Ctrl + I` / `Command + I` でマスクを反転する

### 覚え方

```text
文字を見せたい = そのままマスク
文字部分を抜きたい = マスクを反転
```

# スマートオブジェクトにして変形する方法

## 23. スマートオブジェクトとは

スマートオブジェクトは、元の内容を保ったまま拡大縮小・変形しやすくする機能です。

Adobe公式では、スマートオブジェクトは元画像を直接変更せずに編集でき、変換後も内容を編集できると説明されています。:contentReference[oaicite:3]{index=3}

---

## 24. テキストをスマートオブジェクトにするメリット

| メリット | 内容 |
|---|---|
| 画質劣化を抑えやすい | 拡大縮小を繰り返す作業に強い |
| 複数レイヤーをまとめて変形できる | 文字＋影＋装飾を一緒に変形 |
| 元の文字を編集できる | スマートオブジェクトを開けば中身を直せる |
| フィルターを非破壊で使いやすい | ぼかしなどをあとから調整しやすい |
| ワープや遠近変形に使いやすい | ロゴ風の変形に便利 |

---

## 25. テキストをスマートオブジェクトにする手順

1. レイヤーパネルでテキストレイヤーを選ぶ
2. 右クリックする
3. `スマートオブジェクトに変換` をクリックする
4. レイヤーのサムネイルにスマートオブジェクトのアイコンが付く

---

## 26. スマートオブジェクト化した文字を変形する手順

1. スマートオブジェクト化した文字レイヤーを選ぶ
2. `Ctrl + T` / `Command + T` を押す
3. 拡大・縮小・回転する
4. 右クリックして `ワープ`、`遠近法`、`ゆがみ` などを選ぶ
5. 形を調整する
6. `Enter` / `Return` で確定する

---

## 27. スマートオブジェクト内の文字を編集する手順

スマートオブジェクト化したあとでも、中身を開けば文字を編集できます。

### 手順

1. レイヤーパネルでスマートオブジェクトのサムネイルをダブルクリックする
2. 別ウィンドウで中身が開く
3. テキストレイヤーを編集する
4. `Ctrl + S` / `Command + S` で保存する
5. 元のPSDに戻る
6. 変更が反映される

---

## 28. スマートオブジェクト化の注意点

| 注意点 | 内容 |
|---|---|
| 直接文字編集はできない | 中身を開いて編集する |
| ファイル構造が少し複雑になる | 初心者はレイヤー名を分かりやすくする |
| 完全に自由なピクセル編集には向かない | 必要なら複製してラスタライズ |
| 変形ハンドルが大きく出ることがある | 中身全体の範囲を参照するため |

---

# ラスタライズとの違い

## 29. スマートオブジェクトとラスタライズの違い

| 項目 | スマートオブジェクト | ラスタライズ |
|---|---|---|
| 文字編集 | 中身を開けば可能 | できない |
| 画質保持 | 強い | 拡大で劣化しやすい |
| 変形 | 非破壊で扱いやすい | 直接加工しやすい |
| ブラシ加工 | 直接はしにくい | しやすい |
| 初心者向き | 安全 | 注意が必要 |

---

## 30. ラスタライズする手順

文字を完全に画像化して、ブラシや消しゴムで直接加工したい場合に使います。

### 手順

1. テキストレイヤーを複製する
2. 複製したテキストレイヤーを右クリックする
3. `テキストをラスタライズ` をクリックする
4. 文字が通常の画像レイヤーになる

### 注意

ラスタライズ後は、文字内容を打ち直せません。

必ず元のテキストレイヤーを残しておきます。
