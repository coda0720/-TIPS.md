# 4K素材をHDシーケンスにする
4K素材をズームできる
HDシーケンス上なら4K素材を拡大しても画質劣化しにくい

シーケンス設定でフレームサイズを3840×2160から1920×1080に

## 同じ素材がプロジェクトパネルに何個もある	
編集 → 重複を統合
### 統合されない場合
メディアパス表示で同じ保存場所か確認
フレームレートが同じか
オーディオ、ビデオ情報のチャンネル数や解像度、形式が同じか


# 書き出し設定
## プリセット
ソースの一致 - アダプティブビットレート（中）
## ラウドネス設定
- [ ] ATSC A/85　北米系の放送規格　音量控えめ
- [ ] EBU R128　欧州系の放送規格　音量控えめ
- [x] ITU BS.1770-3　目標LUFSを調整可能、基本測定方式

目標ラウドネス[LUFS]
ｰ24　→　ｰ14
SNS動画 -14
ナレ/インタビュー -14 ~-16
テレビ放送 -23 ~ -24
音楽PV -12前後

許容量[LU]
2 →　1
目標値からどれくらいズレてもＯＫか

最大トゥルーピークレベル[dBIP]
-2  → -1
音が割れないようにする上限
SNS動画　-1
音楽PV　-2

## マスター保存用
Quick Time / Apple ProRes 422 HQ
OR 
DNxHR HQ

## ビデオ
- [x] 最大深度に合わせてレンダリング
- [x] 最高レンダリング品質を使用
