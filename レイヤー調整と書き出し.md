## レイヤー作業でよくある失敗

| 失敗 | 原因 | 対処 |
|---|---|---|
| どのレイヤーを編集しているか分からない | レイヤー名が初期名のまま | レイヤー名を変更する |
| 消しゴムで消したら戻せない | 画像を直接削除している | レイヤーマスクを使う |
| 文字を編集できなくなった | ラスタライズしてしまった | ラスタライズ前のPSDを残す |
| 画像が荒くなった | 拡大縮小を繰り返した | スマートオブジェクト化する |
| 一部のツールが使えない | スマートオブジェクトのまま | 複製してラスタライズする |
| 書き出したら背景が白い | JPG保存、または透明設定がオフ | PNGで透明部分をオンにして書き出す |
| PNGなのにサイズが大きすぎる | 透明範囲や余白を含んでいる | カンバスサイズ・トリミングを確認する |
| 結合後に修正できない | レイヤーを1枚にまとめた | 結合前にPSDを別名保存する |


# Photoshop 2026：グループ化・結合・透明な余白を切り落として書き出す方法

## 1. グループ化と結合の違い

## グループ化とは

**グループ化**は、複数のレイヤーをフォルダーにまとめる操作です。

レイヤー自体は分かれたままなので、あとから個別に編集できます。

### グループ化の特徴

| 項目 | 内容 |
|---|---|
| 役割 | 複数レイヤーを整理する |
| レイヤー構造 | 分かれたまま残る |
| あとから編集 | できる |
| 向いている用途 | 作業中の整理、表示/非表示の管理 |
| 安全性 | 高い |

---

## 3. 結合とは

**結合**は、複数のレイヤーを1枚のレイヤーにまとめる操作です。

結合後は、文字・画像・効果などを個別に編集しにくくなります。

### 結合の特徴

| 項目 | 内容 |
|---|---|
| 役割 | 複数レイヤーを1枚にまとめる |
| レイヤー構造 | 1枚になる |
| あとから編集 | しにくい |
| 向いている用途 | 完成後の整理、処理を軽くしたいとき |
| 安全性 | 低め。実行前に注意が必要 |

---

## 4. グループ化と結合の比較

| 比較項目 | グループ化 | 結合 |
|---|---|---|
| 操作の意味 | フォルダーにまとめる | 1枚のレイヤーにする |
| レイヤーの個別編集 | できる | しにくい |
| 元に戻しやすさ | 戻しやすい | 戻しにくい |
| 作業中の整理 | 向いている | あまり向かない |
| データの軽量化 | あまり変わらない | 軽くなる場合がある |
| 初心者へのおすすめ | 基本はこちら | 慎重に使う |

---

## 5. グループ化の手順

### 目的

複数のレイヤーをまとめて整理する。

### 手順

1. まとめたいレイヤーを複数選択する
2. `Ctrl + G` / `Command + G` を押す
3. フォルダーが作成される
4. フォルダー名を分かりやすく変更する

### ショートカット

| 操作 | Windows | Mac |
|---|---:|---:|
| グループ化 | `Ctrl + G` | `Command + G` |
| グループ解除 | `Ctrl + Shift + G` | `Command + Shift + G` |

### 使う場面

| 場面 | 例 |
|---|---|
| レイヤーが増えたとき | 人物、背景、文字を別々に整理する |
| 一括で表示/非表示したいとき | ロゴ全体をまとめて非表示にする |
| 複数レイヤーをまとめて動かしたいとき | 人物と影を一緒に移動する |

---

## 6. 結合の手順

### 目的

複数のレイヤーを1枚にまとめる。

### 手順

1. 結合したいレイヤーを選択する
2. `Ctrl + E` / `Command + E` を押す
3. 選択したレイヤーが1枚にまとまる

### ショートカット

| 操作 | Windows | Mac |
|---|---:|---:|
| 下のレイヤーと結合 | `Ctrl + E` | `Command + E` |
| 表示レイヤーを結合 | `Ctrl + Shift + E` | `Command + Shift + E` |

---

## 7. 結合するときの注意点

結合すると、文字・画像・効果などを個別に戻しにくくなります。

特に以下のレイヤーは、むやみに結合しない方が安全です。

| レイヤー | 結合しない方がいい理由 |
|---|---|
| テキストレイヤー | 文字を打ち直せなくなる |
| 調整レイヤー | 色補正をあとから変更しにくくなる |
| マスク付きレイヤー | 隠していた部分を戻しにくくなる |
| スマートオブジェクト | 元画像の情報を保ちにくくなる |

---

## 8. 結合する前の安全な手順

結合する前に、必ず編集用データを残しておく。

### 手順

1. 結合したいレイヤーを選択する
2. `Ctrl + J` / `Command + J` で複製する
3. 複製したレイヤーだけを結合する
4. 元のレイヤーは非表示、またはグループに入れて残す

---

# ##透明な余白を切り落として書き出す方法##

## 必要な部分だけPNGで書き出す最短手順

```text
必要なレイヤーだけ表示
↓
背景レイヤーを非表示
↓
イメージ
↓
トリミング
↓
透明ピクセル
↓
上・下・左・右にチェック
↓
OK
↓
ファイル
↓
書き出し
↓
書き出し形式
↓
PNG
↓
透明部分オン
↓
書き出し

## 9. 透明な余白を切り落とすとは

透明背景は残したまま、余計な透明部分だけを削る方法です。

ロゴや切り抜き素材を、見た目の中身に合わせて小さくしたいときに使います。
```

---

## 向いている用途

| 用途 | 内容 |
|---|---|
| ロゴ素材 | ロゴ周辺の余白を削る |
| 切り抜き画像 | 人物や物だけに近いサイズにする |
| EC画像素材 | 余白が大きすぎる素材を整える |
| PNG素材 | 透明背景を残したままサイズを詰める |

---

## 透明な余白を切り落とす手順

1. 書き出したいレイヤーだけを表示する
2. 不要な背景レイヤーを非表示にする
3. 上部メニューの `イメージ` をクリック
4. `トリミング...` をクリック
5. `透明ピクセル` を選択する
6. `上`、`下`、`左`、`右` にチェックを入れる
7. `OK` をクリック
8. 透明な余白が切り落とされる
9. `ファイル` → `書き出し` → `書き出し形式...` をクリック
10. 形式を `PNG` にする
11. `透明部分` をオンにする
12. 必要に応じてサイズを変更する
13. `書き出し` をクリックする

---

## 書き出し時の確認ポイント

| チェック項目 | 内容 |
|---|---|
| 背景レイヤー | 非表示になっているか |
| 透明部分 | 市松模様になっているか |
| トリミング設定 | `透明ピクセル` を選んだか |
| チェック項目 | 上・下・左・右すべてにチェックしたか |
| 保存形式 | PNGになっているか |
| 透明部分 | 書き出し設定でオンになっているか |
| サイズ | 必要なpx数になっているか |

---

## 注意点

この方法は、**カンバス自体の大きさが変わります**。

元の配置や余白を残したい場合は、必ず別名保存してから行います。

### 安全な流れ

1. 元データをPSDで保存する
2. `別名で保存` して書き出し用データを作る
3. 書き出し用データで透明ピクセルをトリミングする
4. PNGで書き出す

---

## まとめ

| やりたいこと | 使う操作 |
|---|---|
| レイヤーを整理したい | グループ化 |
| 複数レイヤーを1枚にしたい | 結合 |
| あとから修正したい | グループ化のままにする |
| 完成後に軽くしたい | 複製してから結合 |
| 透明な余白を削りたい | イメージ → トリミング → 透明ピクセル |
| 透明背景PNGにしたい | PNG + 透明部分オン |
